## Настройка проекта
```
npm install
```
### Компилирует и выполняет горячую перезагрузку для разработки
```
npm run serve
```
### Компилирует и минимизирует для производства
```
npm run build
```
### Выравнивает и исправляет файлы
```
npm run lint
```

[Нажми на меня!](https://cli.vuejs.org/config/).
